* XPService
* ComboService

* local storage
* migration system implemented

## Design Principles

* modular services
* no system overlap
* safe integration between features

## UI Principles

* clear hierarchy
* minimal friction
* consistent feedback
* action-driven interface

## Key Rules

* no blocking gameplay
* systems enhance, not restrict
* feedback must be immediate
* progression must feel visible
