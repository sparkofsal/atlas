# Belief Atlas — Game Systems

## XP System

* Discover item → XP
* Reaction → XP
* Combo → bonus XP
* Goals → bonus XP
* Runs → bonus XP

## Level System

* Progressive curve
* Titles unlock at key levels

## Combo System

* Correct guesses increase combo
* Wrong guess resets combo
* Combo gives XP multiplier

## Streak System

* Daily engagement tracked
* Break after missed day

## Goals System

* 2–3 active goals
* Context-aware generation
* Daily refresh

## Runs System

* Short sequences (2–4 steps)
* Types:

  * country
  * sayings
  * category
  * combo
* Optional but rewarding

## Collection System

* Country-based
* Tracks completion %
* Milestone rewards

## Exploration System

* Play styles:

  * Explore freely
  * New discoveries
  * Finish country
  * Sayings

## Smart Suggestion System

Priority:

1. Level-up proximity
2. Country completion
3. Combo continuation
4. Last mode

## Hybrid Progression System

* No hard locks
* Rarity unlocks by level
* Country depth increases with level

## Continuation System

* Suggest next action after each interaction
* Keep player moving without friction
