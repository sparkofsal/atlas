# Belief Atlas — Product Vision

## Core Idea

Belief Atlas is a discovery-based game where players explore beliefs, sayings, and cultural ideas from around the world while leveling up their understanding of global perspectives.

## Core Fantasy

“I am exploring how the world thinks.”

Players are not just reading content — they are:

* discovering
* judging
* progressing
* collecting

## Experience Goals

The app should feel:

* engaging
* surprising
* smooth
* rewarding
* personal

Players should feel:

* curiosity (“what is this?”)
* judgment (“do I believe this?”)
* progression (“I’m leveling up”)
* purpose (“I want to complete this”)

## Core Loops

### Short Loop

Discover → Guess → React → Gain XP

### Session Loop

Combo → Run → Continue → Progress

### Daily Loop

Daily item → Streak → Return tomorrow

### Long-Term Loop

Country completion → Unlocks → Titles

## Target Audience

* Curious learners
* Casual gamers
* Students
* Global audience

## Tone & Personality

* curious
* slightly mysterious
* playful but intelligent

## Non-Goals

* Not a pure educational app
* Not a social media feed
* Not a complex quiz system
